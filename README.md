# WEEK01
## 결합도(Coupling)
- 느슨한 결합도가 좋다.

💭 Why? 변화에 유연하게 대응하기 위해

### 제어 역전(IoC : Inversion of Control)
> 결합도를 낮추기 위한 방법
- 변화에 **유연하게 대응**하기 위해 **프로그램 제어 흐름의 주도권을 외부**로 넘겨주는 것

#### 외부 주입(DI : Denpendency Injection)
> IoC를 구현하기 위한 방법 중 하나
- 객체가 필요한 **종속성을 외부에서 주입**해 주는 것
